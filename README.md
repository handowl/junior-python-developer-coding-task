Задание для потенциальных кандидатов на позицию Junior-девелоперов.

- входные данные:
  - csv_data_1.csv
  - csv_data_2.csv
  - json_data.json
  - xml_data.xml

- результаты **Basic**:
  - basic_results.tsv

Все входные данные лежат в папке "input_data".
Все выходные дынные будут в "output_data".
